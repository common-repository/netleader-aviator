NetLeader Aviator Guide

NetLeader Aviator computes aircraft weight & balance for pilots using predefined aircraft profiles created by maintenance officers or administrators from official aircraft documents.

NetLeader Aviator Guide for Pilots

Usage:

  1. NetLeader Aviator's opening screen displays a list of predefined aircraft profiles created by your organization. Click on the profile you want to use.
  2. Enter your weights into the Loading Table. Results are automatically recalculated after each entry. Print the results if you wish to save them.

NetLeader Aviator Guide for Officers and Administrators

Users with administrative privileges or roles with the capability of `Edit Dashboard` can create, edit and delete aircraft profiles. Please see your site's webmaster if you need administrative privileges.

Add or Edit a Profile

  1. All data should be taken from the official Pilot's Operating Handbook and FAA Approved Airplane Flight Manual plus the aircraft's most recent weight and balance revision.
  2. To add a new profile, click on 'ADD NEW AIRCRAFT PROFILE' from the opening screen which will open a sample aircraft profile for editing.
  3. To clone an existing profile, edit that profile (see below) and then change the profile's name. A new profile will be created when you save your changes.
  4. To edit an existing profile, click on a profile from the opening screen and then click on the 'Edit Profile' button at the bottom of the screen.
  5. Use the ‘Update Profile’ button at the bottom of the edit screen to save your changes when done.

    6. Your changes will be lost if you do not save them!

Delete a Profile

  1. Edit a profile (see above).
  2. Click on the ‘Delete Profile’ button at the bottom of the edit screen to permanently delete the selected profile.

Editing a Profile

  1. Change the name of the profile, if desired. A profile name should begin with the registration number of the aircraft (N number).
  2. The Summary and CG Envelope sections are computed and cannot be changed.

    3. EXAMPLE: Summary and CG Envelope Sections

  3. Enter each Item’s Description, Weight and Arm in the Loading Table from approved documents for the aircraft. Enter a weight of zero for weights to be entered by Pilots. Moments are calculated and cannot be changed.
  4. The weight of each item containing the word ‘fuel’ (case insensitive) in the description will be used to calculate the Zero Fuel Weight.
  5. Use the ‘Add Row’ and ‘Delete Row’ buttons at the bottom of the Loading Table to add or delete rows at the end of the table.

    6. EXAMPLE: Loading Table

  6. Create the ‘CG Envelope Table’ referencing the ‘Center of Gravity Limits’ published in the aircraft's official Pilot’s Operating Handbook.

    7. EXAMPLE: Center of Gravity Limits Table from aircraft's official documents.

    1. Starting from the lowest weight shown on the CG Limits, enter the Weight, Forward CG and Aft CG for that weight. You may have to convert moment to arm if the limits you are using are a moment envelope.
    2. Enter the Weight, Forward CG and Aft CG for each weight where there is a change in the slope of the envelope.
    3. Enter the Weight, Forward CG and Aft CG for the highest weight shown on the CG Envelope. This weight will be used as the ‘Max Gross’ weight shown in the Summary.
    4. Use the ‘Add Row’ and ‘Delete Row’ buttons at the bottom of the CG Envelope Table to add or delete rows at the end of the table.
    5. Note that only one envelope (e.g. Normal Category) can be entered into a profile. For additional envelopes, create additional profiles.

      6. EXAMPLE: CG Envelope Table

  7. Use the ‘Update Profile’ button at the bottom of the screen to save your changes. A new profile will be created if you have changed the name of the profile to one that is not already in use.

    8. Your changes will be lost if you do not save them!



NetLeader Aviator is for educational purposes only, use official procedures and documents to calculate weight & balance prior to flight.
